We can use Apache running on a linux VM to host websites locally. Apache supports a variety of features, many implemented as compiled modules which extend the core functionality. These can range from authentication schemes to supporting server-side programming languages such as Perl, Python, Tcl and PHP. We can then install Ubuntu (which is a version of Linux) on a virtual machine and set it up to host our website.
